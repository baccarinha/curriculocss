
# 🎨 Currículo com CSS — Currículo Web Estilizado

**Acesse aqui:**
[curriculo](https://baccarinha.github.io/curriculocss/)

🎯 **Objetivo do Projeto**

Criar um currículo digital estilizado utilizando CSS, permitindo apresentar informações profissionais de forma visualmente organizada, moderna e acessível pela internet.

🧩 **Funcionalidades do Projeto**

|O projeto apresenta as seguintes funcionalidades:|
|--|
📄 Exibição de currículo digital
🎨 Estilização visual utilizando CSS
🧠 Organização das informações por seções
📋 Estrutura clara e fácil de ler
🖥️ Acesso direto pelo navegador
📱 Layout estruturado para visualização online
🔗 Compartilhamento fácil por meio de link
🛠️ Tecnologias Utilizadas

|Este projeto foi desenvolvido utilizando:|
|--|
HTML5 — Estruturação do conteúdo
CSS3 — Estilização visual e layout
GitHub Pages — Hospedagem do projeto
Inteligência Artificial (IA) — Apoio na criação da estrutura e organização
